# Home Web

This repository is an minimal webapp describing my Airbnb announcement, built with [Hugo](https://gohugo.io/) using the theme [Hugo story](https://github.com/caressofsteel/hugo-story)


# Demo

[https://juno-coca.netlify.app/](https://juno-coca.netlify.app/)


# Using 
1. [Install Hugo](https://gohugo.io/overview/installing/)
2. Clone this repository

```bash
git clone https://github.com/Moq77111113/home-web.git
cd home-web
```

3. Run Hugo

```bash
hugo server
```
